Dự án học phần **Lập trình Web nâng cao (EPU)**. Hệ thống giúp quản lý danh mục tài sản, theo dõi vòng đời thiết bị và điều chuyển tài sản trong tổ chức một cách hiệu quả.

---

## ✨ Tính năng chính

- **Quản lý tài sản:** Thêm, sửa, xóa và tra cứu thông tin tài sản (máy tính, thiết bị văn phòng, phần mềm...).
- **Theo dõi vòng đời:** Ghi nhận lịch sử bảo trì, khấu hao và thời gian thanh lý thiết bị.
- **Quản lý điều chuyển:** Theo dõi việc bàn giao tài sản giữa các phòng ban hoặc cá nhân.
- **Báo cáo & Thống kê:** Xuất dữ liệu thống kê tình trạng tài sản dưới dạng bảng biểu.
- **Phân quyền người dùng:** Quản trị viên (Admin) và Nhân viên (User) với các quyền hạn khác nhau.

---

## 🛠️ Công nghệ sử dụng
A
- **Backend:** ASP.NET Core 8.0 (MVC)
- **Database:** SQL Server (hoặc SQLite cho môi trường Dev)
- **ORM:** Entity Framework Core (Code First)
- **Frontend:** Bootstrap 5, jQuery, DataTables
- **Công cụ:** Visual Studio 2022, Git
